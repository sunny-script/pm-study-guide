HTML

  • ! + Enter is a shortcut to create HTML skeleton in VS Code.
  • The head element contains information about the webpage - not visible to the user.
  • The body element represents the visible content shown to the user.
  • The main element contains the most relevant page info. E.g. exclude headers, footers, nav bars, etc.
  • Attributes can be added to the opening tag of an HTML element to modify its behavior or appearance.
  • Semantic HTML tags convey the meaning fo the content within them. E.g. header, article, footer are, while div and span are not.

GIT

  • git status: checks what branch currently on. shows status/errors.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git checkout branch-name: navigates to specified branch.
  • git pull origin main: pulls from main branch
  • git fetch: makes git aware of other branches
  • git init: initializes git repo.
  • git add .: adds change, always do prior to commiting.
  • git commit -m "message": commits/saves
  • git remote add origin past-copied-link-from-github: add repo.
  • git push -u origin main: pushes to main branch.
